# Santader

Este é um projeto de uma pagina do santander,criado somente para pratica de conceitos web.

## Rodando projeto
> Clone o projeto do github 
$ git clone https://github.com/Leandro-Faustino/Interface-do-banco-Santander.git <br>

> Baixe as dependências
$ npm install <br>

> Para rodar:
$ ng serve -o <br>
